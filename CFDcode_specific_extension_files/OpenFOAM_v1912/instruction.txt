
Waves2foam
==========


1. download waves2foam.
2. go into folder: ".\waves2Foam\src\waves2Foam\waveTheories\externalWaveForcing\" and create a folder called CFDwavemaker
3. copy the "CFDwavemaker_OF1912.C" and "CFDwavemaker_OF1912.H" into the new directory together with the precompiled library "libCFDwavemaker.a"
4. add the CFDwavemaker folder to make file setup in: ".\waves2Foam\src\waves2Foam\Make\files

<-------------------------------------------------------------------->
/* External wave source */
extWF=externalWaveForcing
$(waveTheories)/$(extWF)/externalSource.C
$(waveTheories)/$(extWF)/externalWaveForcing.C
$(waveTheories)/$(extWF)/emptyExternal/emptyExternal.C
$(waveTheories)/$(extWF)/oceanWave3D/oceanWave3D.C
$(waveTheories)/$(extWF)/CFDwavemaker/CFDwavemaker_OF1912.C
<-------------------------------------------------------------------->

5. in the options file, add the CFDwavemaker library to list: ".\waves2Foam\src\waves2Foam\Make\options

<-------------------------------------------------------------------->
LIB_LIBS = \
    -lfiniteVolume \
    -lmeshTools \
    -lsampling \
    -lfileFormats \
    -lstdc++ \
    -lgfortran \
    -L$(WAVES_LIBBIN) \
    -lOceanWave3D \
    -lCFDwavemaker \
    -L$(WAVES_GSL_LIB) \
    -lgsl \
    -lgslcblas
<-------------------------------------------------------------------->

6. go to .\waves2Foam and run ./Allwmake. Be sure to check that the installation directory is in accordance with you OF1912 installation.
7. waveFOAM will now be created including the CFDwavemaker library.
